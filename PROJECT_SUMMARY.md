## 🔧 Installation & Setup

### Prerequisites
- Python 3.8+
- pip package manager
- Modern web browser

### Quick Setup
```bash
cd bs_pricing_model
python3 setup.py          # Automated setup
pip3 install -r requirements.txt  # Install dependencies
python3 app.py            # Start web application
```

### Access Application
Open browser to: `http://localhost:5000`
